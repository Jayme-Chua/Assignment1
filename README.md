# Sem2_FED_Assignment1

Theme: Website for TheFatRat

## Design Process
Wireframe link: https://www.figma.com/file/DTFsR0zAmPgVsIICszgMYg/FED_Assg1?type=design&mode=design&t=x6GptsvOgWUWdfUs-1
<br>This website was made for fans of the artist TheFatRat who would like to know more about the artist and find out any information of events coming up. Music videos are also displayed for easier access for the fans

## Features
Feature 1: Allows fans to navigate to the artist's social media pages by clicking on the icon. Links open in a new tab
<br>Feature 2: Image slideshow in index.html to cycle through images of the artist using JavaScript.
<br>Feature 3: A form in tours.html to save user's information in local storage and display user's name.
Validation is done to ensure user's email is in the correct format using JavaScript.
<br>
<br>Features left to implement: A shop and cart for the artist's merchandise
